# 02 — Format Specs
> Exact production rules per format. Agents reference these; Canva/Higgsfield execute them.

---

## Universal rules (all formats)

- Watermark `@thevelum.studio` on **every slide/frame** — distribution seal, not CTA. DM Sans 300, low opacity.
- { • } mark top-left on covers and closing slides.
- Typography per Constitution v5: DM Serif Display headlines, one gold italic accent word, DM Sans everything else. No Playfair.
- Gold never exceeds ~8% of any composition. Never solid gold blocks or tags.
- Aspect ratios: feed 4:5 · reels/stories 9:16 · grid/profile assets 1:1.

---

## Carousel (photo-driven system — LOCKED)

**Structure — 5 to 7 slides:**

| Slide | Ground | Content |
|---|---|---|
| 1 (cover) | Full-bleed working-moment photo + navy semi-transparent overlay | Eyebrow + dominant headline (max 6 words, one gold italic word). { • } top-left. No body copy. |
| 2, 4, 6 (even) | Navy solid or Clear — pure typography | One claim per slide: eyebrow + headline + 2–3 sentences DM Sans 300 body max. |
| 3, 5 (odd) | Full-bleed photo + navy overlay + text | Short headline or single-sentence statement over image. |
| Closing | Always navy | Brand mark { • } + wordmark + the single CTA + footer standard. |

**Rules:** single dominant headline per slide · one idea per slide · photography from the working-moments library only · alternation is strict — never two photo slides adjacent.

*(The v4 "split editorial" half-photo/half-navy layout is legacy for carousels; it remains valid for landing pages and proposals.)*

## Statement post (single image)

- Navy ground, subtle dot-grid texture (~3% opacity)
- Eyebrow: gold dash + DM Sans 700 uppercase
- Headline: DM Serif Display, large, off-white, max 6 words, one gold italic word — must create tension standalone
- Context note: one sentence, max 15 words, bottom-right, small italic — adds precision, never explains the headline
- Footer standard bottom center

## Reel (non-Vera)

- Length: 15–30s. Faceless B-roll (working moments) + text overlays.
- **Overlay typography:** DM Serif Display for the key line, DM Sans for supporting lines. Max 5 overlay beats.
- Beat structure: `HOOK (0–3s, on-screen text = the problem) → 2–3 CLAIM BEATS → CLOSE (brand mark + one CTA)`
- Audio: low-key, editorial. Never trending-audio-chasing, never voiceover hype.
- Caption: 2–4 sentences expanding the argument (not repeating overlays) + one CTA + footer standard.
